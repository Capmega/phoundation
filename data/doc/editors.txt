The editors library currently ony has the tinymce editor implemented

The tinymce editor has the jbimages images uploader plugin bundled.
pub/js/base/tinymce/plugins/jbimages/config.php contains configuration,
but all configuration is setup so that it can be set by correctly configuring
$_CONFIG[editors][images]
